Simulating (Almost) Linear Models

Charles Zheng

2015-05-02

Introduction

Linear model:

\[ y = x_A '\beta + \epsilon \] where \(x_A\) are active variables,

\[ x_I = x_A' \Gamma + E \]

where \(x_B\) are inactive variables.

Almost-linear model:

\[ y = x_A' \beta_A +\delta f(x_A) + \epsilon \]

while as before,

\[ x_I = x_A' \Gamma + E \]

where \(f(x_A)\) is a nonlinear perturbation satisfying \(\text{E}[f(x_A) x_A] = 0\)

In both the linear and almost-linear model, if we define \[ \beta = \text{E}[x x']^{-1} \text{E}[y x] \] then \[\beta_I = 0\] and \[\beta_A = \beta_A\]

Examples of nonlinear functions

Nadaraya-Watson surface:

Let \(u_1, .., u_k\) be control points in the space of active variables \(\mathcal{X}_A\), let \(\alpha_1,...,\alpha_k\) be real-valued amplitudes and let \(w_1,..., w_k\) be weights, and \(\phi(x)\) the standard multivariate normal density Then define \[ \tilde{f}(x) = \frac{\sum_{i=1}^k w_i \alpha_i \phi(x - u_i)}{\sum_{i=1}^k w_i \phi(x - u_i)} \] and \[ f(x) = \tilde{f}(x) - \gamma'x \] where \(\gamma\) is defined by \[ \gamma = \text{E}[x x']^{-1} \text{E}[\tilde{f}(x) x] \]

Inference

Linear case

Generate design matrix, true beta, and nonlinear perturbation

library(reginference)
library(magrittr)
p <- 10
n <- 1000
sigma_y <- 3
bt <- rnorm(p)
x <- randn(n, p, colnames = TRUE)
y0 <- x %*% bt
noise <- sigma_y * rnorm(n)
y <- y0 + noise

Generate derived variables

x_aug <- noised_projections(x, sigma = 0.1, q = 100, adjoin = TRUE)
colnames(x_aug)[1:20]

Logistic transform case

lf <- function(x, delta) { 2/delta * (exp(x * delta) - 1)/(1 + exp(x * delta)) }
delta <- 1
yn <- lf(y0, delta) %>% t %>% t + noise
